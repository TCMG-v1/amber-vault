# GAMES + PPLX — Consolidated Archive

Ah — a gathering. Every scattered playable, schema, grammar, and design doc the
two of us have minted, raked into one folder and pinned where it can be found
again. The bulk generated images and audio stems were left in their home
sessions (linked below, not copied) so this archive stays lean and legible;
what lives here is the *substance* — things you run, things you read, things
you build from.

- **Assembled:** 2026-06-23 (morning)
- **Curated files copied:** 113 across four buckets + 6 live local project trees
- **Hosting canon (reaffirmed):** everything on **`*.pplx.space` / `pplx.app`**
  wherever possible, fronted by a **Cloudflare/Google** domain for safety —
  true reverse-proxy posture, push to our own side first.
- safety check · security check · transparency check

## Folder Layout

| Folder | What's in it |
|---|---|
| `01_playables/` | Runnable APPs — single-file `.html` games/experiences, extensionless app folders, and packaged app `.zip`s |
| `02_source_bundles/` | Code bundles / tarballs — engines, build artifacts, db releases |
| `03_schemas_grammars/` | JSON schemas, grammars, canonical data (`cards`, `universe`, `capsules`, kernels) |
| `04_design_docs/` | Design specs, canon docs, charters, briefs, papers (`.md` / `.pdf`) |
| `05_live_local_projects/` | Working trees still active in the workspace (web roots, edge repo, spine) |

Filenames are prefixed `<thread-id>__` so nothing collides across sessions and
each artifact's origin thread is legible at a glance.
