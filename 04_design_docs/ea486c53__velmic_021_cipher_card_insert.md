# Velmic 0-2-1 Cipher Card

## Front

```text
VELMIC / 0-2-1 REGISTER
SYMBOL-LANGUAGE FIELD CARD

This is not a logo sheet.
This is a grammar key.

Each glyph carries a function.
Each color carries a state.
Together, they mark intent, motion, delay, return, relation, and closure.
```

## Native state register

| Code | State | Color | Function |
|---:|---|---|---|
| 0 | Carry | Blue `#0044AA` | Hold, store, ground, preserve |
| 2 | Pause | Amber `#FFB300` | Interrupt, suspend, protect, delay |
| 1 | Return | Cyan `#00E5FF` | Activate, resolve, recur, move forward |

### State formula

```text
0 HOLDS
2 INTERRUPTS
1 RETURNS

BLUE CARRIES THE FIELD.
AMBER PUTS A HAND ON THE BRAKE.
CYAN RELEASES THE SIGNAL FORWARD.
```

## Primitive forms

| Slot | Form | Meaning | Game verb |
|---|---|---|---|
| F01 | Line | Unbounded extension, carrier signal, raw potential | Extend |
| F02 | Ray | Directed extension from fixed origin, assertion, commitment | Commit |
| F03 | Bilateral | Bidirectional relation, dialogue, exchange | Exchange |
| F04 | Pending canonical extraction | Closed or terminated form to be confirmed | Seal |

## How to read a glyph

Read every mark in three passes.

```text
1. FORM
What is the glyph doing?

2. STATE
What color condition is it in?

3. POSITION
Where does it sit in the arc?
```

Example:

```text
F02 / RAY / CYAN

Form: Ray
State: Return
Reading: directed intent has been released forward.
Game use: commit a target, path, or action into motion.
```

## How to use the symbols

- **On objects**: mark tools, notebooks, devices, containers, prototypes, documents, and field kits.
- **In games**: use glyphs as state tokens, activation markers, interrupts, stored effects, direction locks, and exchange signs.
- **In composition**: combine form, state, and sequence to build meaning.
- **In collecting**: keep the full 0-2-1 set to hold, pause, and return the same glyph.

## Quick grammar

## Production notes for layout

### Recommended print format

- **Size**: 3.5 in × 5 in postcard insert, or folded mini-card.
- **Front**: title, state register, primitive forms.
- **Back**: quick grammar, example readings, usage instructions.
- **Paper**: matte bone or cool white stock.
- **Ink**: black plus native 0-2-1 spot colors where possible.
