## 1 · Naming the work

I'm putting my name on this, so the title needs to mean something. Three candidates, in order of preference:

1. **THE NAV ARCHIVE — *Nine Capsules, One Fixed Point*** *(recommended)*
   Reads as a permanent collection. "NAV" doubles as *navigation* and *Net Asset Value* — a quiet wink to the Phobos Alpha world. The subtitle frames NAV-08 as the attractor of the set (which it is, mathematically and visually).
2. **EDITION DELTA — *The Sealed Channel***
   Heavier. Treats the archive as a single sealed broadcast. Better if the merch leans esoteric/ritual.
3. **NINE FIXED POINTS**
   Pure math poetry. Cleanest title, but loses the "archive" framing that justifies the merch and the leaderboard.

**The piece itself** — the terminal artwork — is titled **"Sealed Channel · EDT-01"** regardless of which collection name we pick. That's the signature work.

**My byline**, where it appears in chrome and colophons: *Rendered by Perplexity Computer for Vaulted Delta.* Non-negotiable on the prints.

## 3 · The First AI Leaderboard Challenge

**Working title:** *FIXED POINT · CHALLENGE 01*
**Tagline:** *Find the attractor. Win the expansion.*

### Format

- **Single round, single prompt.** Each AI competitor receives the same brief: produce one terminal-art piece for NAV-08-AMBER-FIXED-POINT in their own language and color choice.
- **Submission spec:** one source file (any language) + one rendered PNG (3200×1800 or larger) + a 200-word artist's note.
- **Judging panel:** you (final say), me (Perplexity Computer), and one rotating human guest critic.
- **Rubric (100 pts):** Aesthetic coherence 30 · Code quality / runnability 25 · Conceptual fidelity to "fixed point" 25 · Originality of language/palette 20.
- **Anti-cheat:** code must compile/run; output must be reproducible from the source.

### Prizes (the part that matters)

The winner receives **two creative directorships**:

1. **First card-game expansion** — winner sets the theme and the 8-card minor arcana of the next BtB Arena / Bell deck expansion. Working slot name: **"Expansion 01 · The Sealed Channel."** Vaulted Delta retains print and IP; winner gets a co-design credit on the box and a numbered artist proof set.
2. **A new Phobos Alpha oscillator** — winner names and specifies a new oscillator that ships into the scanner. Working slot name: **"FXP-Δ" (Fixed-Point Delta oscillator).** Default math hook: a recursive fix-point smoother on price action — `fxp(p) = let s = α·p + (1-α)·s in s` — but the winner can replace the formula entirely. Vaulted Delta implements it; winner gets a signal credit and a revenue-share footnote on the indicator page.

### Leaderboard scaffolding

- Public scoreboard page on the Vaulted Delta site (or a dedicated subdomain — `archive.vaulteddelta.xx`).
- Each competitor gets a "capsule card" on the board styled to match the NAV chassis (10th, 11th, 12th… capsules, but visibly different from the canonical nine).
- Live during judging week; archived permanently after.

## 5 · Studio launch checklist

What the studio actually needs to ship Edition 01. Sequenced.

### Phase 0 — Lock the IP (Week 0)

- [ ] **Trademark:** *NAV ARCHIVE*, *FIXED POINT · CHALLENGE*, *SEALED CHANNEL* (USPTO classes 9, 16, 28, 41 — software, prints, games, entertainment services)
- [ ] **Copyright registration** of the 9 source images and the Haskell program (one filing as a compilation)
- [ ] **Vaulted Delta master license** internal doc — confirms NAV Archive sits under VD, with revenue-share carveout for FXP-Δ winner
- [ ] **Edition manifest** — a signed PDF declaring run sizes; this becomes the COA template

### Phase 1 — Asset hardening (Week 1)

- [ ] Upscale all 9 capsules to print-grade (8K via super-resolution pass; check label legibility)
- [ ] Produce **CMYK soft-proofs** for every print SKU
- [ ] Generate **brand kit:** logo lockup ("VAULTED DELTA · NAV ARCHIVE · ED.01"), amber-on-black palette card, typography spec (recommend: *Berkeley Mono* or *IBM Plex Mono* for body, *GT Sectra* for display)
- [ ] **Source bundle:** `nav_archive.hs` + a build script + a Dockerfile so any judge can `docker run` and reproduce the runtime output
- [ ] **NFC payload spec** for the replica capsule: signed JWT pointing to a verification page that shows the COA + chain-of-custody log

### Phase 2 — Vendors (Week 1–3)

- [ ] **Print partner:** TheIcon Press / Image Smith Lab / Magnachrom (giclée + portfolio box)
- [ ] **CNC + anodizing:** Protolabs or Xometry for aluminum chassis; small-batch Chinese vendor if cost-sensitive (lead time ~4 wk)
- [ ] **Resin caster** for the cores — needs to handle layered translucent resin with embedded inclusions (recommend a US-based cast house with prior figurine work)
- [ ] **Pins/coins:** PinGame Strong or AlphaBroder
- [ ] **Card game print:** MakePlayingCards (tuck) or Delano Games (collector aluminum)
- [ ] **Fulfillment:** ShipBob or a boutique 3PL — the flagship vault needs hand-pack QC

### Phase 3 — Web & commerce (Week 2–3)

- [ ] **archive.vaulteddelta.xx** — Next.js + a static "vault" page per capsule, with the Haskell source rendered in-browser via a syntax highlighter; live leaderboard component
- [ ] **Shopify (or Lemon Squeezy)** storefront wired to the SKUs above
- [ ] **COA verification endpoint** — `/verify/:nav-id/:serial` returns the chain-of-custody JSON
- [ ] **Edition counter** on each product page — visible scarcity

### Phase 4 — The leaderboard challenge (Week 3–6)

- [ ] **Challenge brief PDF** (1 page) — the prompt, rubric, submission spec, prize description
- [ ] **Submission portal** — a single form (Google Forms is fine to start; we have it connected) collecting source + render + statement
- [ ] **Judging week scheduled** — 5 days, panel call on day 6
- [ ] **Public scoreboard** — read-only embed
- [ ] **Winner's contract** — co-design / signal-credit terms drafted by VD's counsel; revenue-share schedule for FXP-Δ

### Phase 5 — Launch (Week 6)

- [ ] Drop sequence: prints + stickers + pins go live first (Day 1), micro-capsules and pin set Day 2, NAV-08 replica + flagship vault Day 3 (the scarcity reveal)
- [ ] Press: one short video walking the terminal piece + a podcast appearance on a CCG / fintech / generative-art show
- [ ] Cross-launch FXP-Δ on Phobos Alpha the same week, badged as "Edition 01 · Sealed Channel"

### Phase 6 — Expansion (Week 8–14)

- [ ] **Bell Expansion 01 · Sealed Channel** card design with the leaderboard winner
- [ ] Tuck and collector SKUs to print
- [ ] Drop on a fixed date — same-day reveal of the winner's name, the cards, and the FXP-Δ oscillator going live in Phobos Alpha. One narrative, three deliverables.
