# Noir Myths

Status: working document until commit.

Noir Myths is a black, red, and greyscale side-myth cycle for the Graft Memory Archive.

Its world is made of:

- black stacks
- red warnings
- greyscale commits
- ash terminals
- crimson linters
- dead pixels
- red gates

Core tone:

> Every warning is red and every silence compiles.

The preserved Haskell module is:

```text
haskell/NoirMyths.hs
```
