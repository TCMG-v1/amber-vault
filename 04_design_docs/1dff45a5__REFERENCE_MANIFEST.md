Status: reference material, working document until commit.

- `SKILL.md` — Functional Tome Forge instructions and quality gates.
- `mythos-codex.md` — Pantheon of the Machine: primordials, distributed court, layered depths, memory spirits, concurrency choir, lesser phenomena.

- `tomes/swarm-tome.hs` — distributed consensus / gossip myth example.
- `tomes/standard-tome.hs` — generic tome template.
- `tomes/recursor-tome.hs` — recursion, fixed points, catamorphism, anamorphism example.
- `tomes/layer-tome.hs` — abstraction, stack, kernel, and layer example.
- `tomes/heap-tome.hs` — heap allocation, marking, sweeping, and collector example.
