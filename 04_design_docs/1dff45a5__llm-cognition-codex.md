# LLM Cognition Codex

`LlmCognitionCodex.hs` is a working module for classifying AI-style model archetypes without binding the project to real vendor names.

It maps model behavior into:

- `Color` — trust, risk, and operational feel
- `DomainFocus` — intended use domain
- `ArchitectureClass` — structural design constraint
- `AlignmentLevel` — guardrail profile
- `CapabilityMatrix` — tool use, context, multimodality
- `LLMModel` — full archetype record
- `MythicEnvelope` — bridge into Graft Memory Archive roles

## Neutral archetypes

- Aurora — general reasoning, high guardrails
- Helios — multimodal fusion, frontier-leaning
- Verdant — open-source general reasoning
- Ember — creative narrative, higher variance

## Archive rule

Model cognition remains a working document until policy, role, validation, and memory agree.
